Usefull vim configuration

When working with Vim here is a useful configurations.

1) Create a .vimrc in the ~/ (User's home directory) 2) Add the line "set nocompatible". to .vimrc 3) Instead of entering Ctrl-e you can you tab for command line completions.

This is often the first recommended configuration to add to vim because it prevents using old fashion Vi things. Vim came after vi from the early days of text editing. It stands for "Vi Improved" allowing user to work in a way that is improved. This example I show, makes completing filenames, paths, internal commands, simple with one one keypress. Consider how it enables command line mode history as well, I almost always use that feature.
